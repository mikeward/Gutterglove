Colt

Free Wordpress Theme by NattyWP

Installation

  • 1. Primary: Login to your wp-admin area and go to Appearance -> Themes. Select Install tab and click on Upload link. Select theme .zip and ckick on Install now button. If you have any errors, use alternate method.
  • 2. Alternate: Unzip the template file (delicate.zip) that you have downloaded. Via FTP, upload the whole folder (Delicate) to your server and place it in the /wp-content/themes/ folder. Do not change directory name. The template files should thus be here now: /wp-content/themes/delicate/index.php (for example).
  • 3. CHMOD directory cache (wp-content/themes/delicate/cache) to 755 permissions.
  • 4. Log into your WP admin panel and click on the Design tab. Now click on the Delicate theme to activate it.
  • 5. Complete all of the required inputs on the Delicate Options page (in the WP admin panel) and click "Save Changes".
  • 7. Read our Knowlegebase
